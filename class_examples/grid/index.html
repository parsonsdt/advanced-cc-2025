Justify-Items (right)

1
2
3
4

*The justify-items property is set on the grid container to give child elements (grid items) alignment in the inline direction.

*You can set the justify to the either of the values normal | stretch | start | left | center | end | right and others

*place-items sets both the align-items and justify-items properties in a single declaration.
